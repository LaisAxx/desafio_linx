## Tecnologias Utilizadas
- HTML5
- CSS3
- Sass 1.35.1


## Como Utilizar

instale o Sass

`npm install sass`


## Servidor de Desenvolvimento
instale o Live-Server para executar a página no seu navegador web. O live-server é um pequeno servidor de desenvolvimento.

`npm i live-server`
